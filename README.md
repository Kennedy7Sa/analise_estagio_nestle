# 📊 Projeto de Análise de Dados — Prova Excel (Estágio Nestlé)

## 📌 Sobre o Projeto
Este projeto foi desenvolvido como parte de uma prova prática de análise de dados utilizando Python, com base em uma planilha Excel contendo informações de vendas, produtos e clientes.

O objetivo foi simular análises que normalmente seriam feitas no Excel, porém utilizando **Python + Pandas**, demonstrando maior eficiência e escalabilidade no tratamento dos dados.

---

## 🗂️ Estrutura dos Dados

O arquivo `Prova+Excel.xlsx` contém 3 abas principais:

### 🔹 Base Vendas
- `SKU`
- `Qtd Vendida`
- `Loja`
- `Data da Venda`
- `Código Cliente`

### 🔹 Cadastro Produto
- `SKU`
- `Produto`
- `Marca`
- `Tipo do Produto`
- `Preço Unitário`
- `Custo Unitário`

### 🔹 Cadastro Clientes
- `Código Cliente`
- `Nome Cliente`
- `Sexo`
- `Nº de Filhos`
